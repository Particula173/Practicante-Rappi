# Practicante-Rappi
### Configuración de API Key

Este proyecto utiliza la API de Groq para el chatbot.

Ve a a api grock y crea una key tu mismo. No te preocupes emplee un modelo gratuito

Luego  de colaner el repositorio ve al explorador de arhivos donde esta elproyecto y abre la terminal del proyecto y pones:

'set GROQ_API_KEY=tu_api_key_aqui'
### Dashboard
Para ejecutar el dashboard o la app pones en la misma terminal:

'streamlit run app.py'

Tambien debes instalar las siguientes libreria por si no las tienes:

'pip install pandas numpy matplotlib streamlit langchain langchain-groq python-dotenv'
